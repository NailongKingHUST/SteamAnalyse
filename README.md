## 整体曲线

### 可通关式游戏

（e.g. 《之狼》《大镖客》《黑神话：悟空》），游戏在线人数应该大致为：1) 开服在线人数短期内达到峰值；2) 从峰值开始，由每个极大值点拟合的曲线呈类反比例函数 __（有待商榷）__ 姿态；3) 将时间尺度无限放大后，游戏在线人数趋于稳定，在某个数值上下浮动；

### 可持续运营式游戏

（e.g. 《APEX》《CSGO2》《...（待补充）》），由于运营时间较长，巅峰人数数据不可查，但是游戏总是在一个最大值和一个最低值之间波动；

### 共同特点

共同特点是在线人数总是高低浮动，该特征反应了玩家的普遍作息规律（注意 __不同地区时差的不同__，从这一点还能反映不同地区游戏玩家的数量），不过可以着重分析一下 __节假日、周末或者版本更新等__ 特殊时间；

__注意__：大数据体现在玩家数量和时间尺度，而不是分析的游戏的数量。在有大量玩家基数下，分析游戏在线人数体现的玩家的群体特征，例如，针对 `1.`，可以分析的是玩家数量整体呈下降趋势，说明随着通过人数增加，在线人数会降低；对 `2.` 也有类似的点可以进行分析。

## 以七天为周期的傅里叶变换

通过这种处理, 我们发现整体趋势偏向于周六周日. 也就是说, 玩家更倾向于在周末打游戏. ~~废话~~

那么一个游戏越倾向于在周末在线人数更多, 这个游戏就被我们认为越 **班味**. 所以这次我们分析的游戏中, 班味之王就是:

- Dragon's Dogma 2

<!-- 2054970 -->

观察评论语言, 这个游戏的主要受众是东亚国家, 这也恰恰反映了东亚的社会压力.

- 永劫无间

<!-- 1203220 -->

但是令人费解的是, 刚刚得出了这个结论, 最不班味的游戏的玩家却也是东亚人.

我们结合这种游戏的特点, 推测这是因为这类游戏为了用户粘性, 会推出很多每日任务, 签到一类的活动, 使得用户养成每天登录的习惯. 又因为我们的算法只关心每一天的峰值, 所以不受用户实际游玩时间影响.

- 黑神话悟空

<!-- 243750 -->

取向很小, 大体原因有两个推测: 第一是受众群体多为中国学生, 而发售时正值暑假, 所以最大权重的一部分是对日期不敏感的. 第二是对于这种游戏, 玩家热情较高, 通关意愿强烈, 所以周几只会影响玩家的游玩时间, 而与玩家当天是否启动这个游戏关系不大.

<!-- 1818750 -->

但是又发现取向稍偏向工作日, 因此推测更新更多存在于工作日.

### 整体观察

总体而言，在一周尺度下，每日周期波动受工作日与周末影响较小的游戏大多是一些多人在线竞技类游戏，这些游戏或是已经培养了玩家的游玩习惯，作为日常游玩，或是需要用户每日登录打卡刷道具材料，大多会有充值获取概率道具的玩法，以此来保证玩家活跃与游戏活力，此类游戏希望玩家保持对游戏的依赖性，玩家长期游玩才能保证游戏有长久的收益，以此会有经常性的活动与更新，引入PVP鼓励引导充值竞争
